##### v3.0.0 「Agent 对话系统 — 让白守拥有灵魂」

> **日期**：2026-03-15 ~ 2026-03-28
>
> **里程碑**：白守从"记录工具"进化为"有记忆的 AI 伴侣"。完整实现了 Agent 对话引擎、日记工具生态、三表存储架构、Token 计费、滑动窗口、AI 伙伴系统、RAG 语义记忆、会话压缩引擎、多工作区隔离架构、网络搜索工具链、MCP 协议接入以及全平台深度稳定性修复。历时两周密集开发，提交超 260 个 commit，由 Anson 与樱＆晓协作完成。

**Phase 1 — Agent 运行引擎搭建**

- **Agent 子系统骨架**：从零搭建 `AgentRunner` 核心循环、`AgentEvent` 事件体系（TextDelta / ToolCallComplete / AgentComplete）、`AgentConfig` 配置。支持流式文本输出、多工具串行/并行执行、最大步数限制与错误恢复。
- **三大供应商 Token 用量解析**：OpenAI（stream_options include_usage）、Gemini（usageMetadata）、Anthropic（message_start/delta）全部接入 `TokenUsage` 模型，支持 input / output / cached 三维度统计。
- **日记工具生态**：实现 `diary_read`、`diary_list`、`diary_search`（FTS5 + 日期范围过滤 + snippet）、`diary_write`、`diary_edit`、`summary_read` 六个工具，Agent 可自主调用读写用户日记。
- **SystemPromptBuilder**：参数化 persona / guidelines / 时间上下文 / 可用工具列表，构建完整的系统提示词。
- **三表存储架构**：重构数据库为 Session → Message → Part 三层结构。Part 级 JSON 存储实现 TextPart / ToolPart / StepFinishPart / CompactionPart 的细粒度追踪。独立 `agent.db`（drift ORM，schemaVersion=7）。
- **Token 计费与 200K+ 阶梯定价**：`ModelPricingService` 从 models.dev 拉取价格（1h 缓存），micro-dollar 精度计费。input + cached 超 200K 自动切换 `context_over_200k` 阶梯价格。费用累加存入 AgentSessions 表。
- **滑动上下文窗口**：`ContextWindow` 服务取最近 N 条消息（默认 30，用户可配，最小 5 无上限），不截断 tool call/result pairs，兼顾省 token 与上下文完整性。

**Phase 2 — AI 伙伴系统**

- **伙伴 CRUD**：实现 emoji 头像 / 图片头像 / 名称 / 简介 / 系统提示词的完整管理闭环，含管理页面、编辑页面与聊天页内 Picker 选择弹窗。
- **默认伙伴机制**：首次启动时自动创建"白小守"默认伙伴（🍵）；至少保留一个伙伴的保护机制。
- **伙伴级配置**：模型绑定（覆盖全局默认模型）、上下文窗口配置（有限 / ∞ 无限）、压缩参数（阈值 + 保留互动轮数）。
- **拖动排序与搜索**：伙伴列表支持拖动排序、搜索过滤，桌面端双栏弹窗包含可编辑标签面板。
- **伴侣模式的生死**：先新增 `companionMode` 固定对话会话（❤ 心形标识），后经评估全面移除，统一为纯会话模式。
- **全局术语改名**：将全系统"助手 (Assistant)"统一重命名为"伙伴 (Partner)"，四语同步。

**Phase 3 — 会话压缩引擎**

- **CompressionService**：基于当前上下文 inputTokens 精确触发（非历史累加），通过压缩提示词模板生成摘要注入上下文。
- **伙伴级 compressKeepTurns**：独立控制每个伙伴的压缩保留互动轮数。
- **数据库迁移**：v4 新增 `compressKeepTurns`、v5 新增 `sortOrder` 拖动排序。

**Phase 4 — RAG 语义记忆**

- **向量搜索核心**：切换至原生 `sqlite-vec` 向量引擎，cosine 距离 + L2 归一化 + HybridSearch（FTS5 关键词 + 向量语义 + RRF 重排序融合）。
- **Embedding 模型管理**：自动检测嵌入维度；异步嵌入迁移（备份表方案 + 崩溃恢复 + 完整性校验）；嵌入重试机制（3次指数退避 + 网络波动容错）。
- **记忆管理页**：统计面板 + 一键清空 + 全文查看 + 条目独立编辑/删除 + 异构模型探测提醒 + 手动迁移入口。
- **日记自动嵌入**：在文件监听层统一管理日记保存时的增量向量同步，引入 `source_created_at` 分离时间语义，chunk 扩展至 1024/128，注入日期/标签前缀。
- **记忆存储工具**：Agent 主动存储重要信息 + 记忆去重与合并服务。
- **RAG 开关联动**：关闭时自动排除语义搜索和记忆存储工具。RAG TopK 上限放宽至 100。
- **网络搜索向量存储**：搜索结果自动向量化压缩存入 RAG，供后续对话引用。
- **数据库重构**：v6 迁移至 `memory_embeddings` 统一表结构，v7 新增 `source_created_at`。

**Phase 5 — 网络搜索工具链**

- **双引擎搜索**：DuckDuckGo Lite + Bing/Google 内置搜索 + Tavily API 付费搜索。
- **Multi-Query 去重**：拆分用户查询为多个搜索词组合，合并去重结果。
- **反爬策略**：UA 池随机切换、HTML→Markdown 清洗、大页面 OOM 防护、纯文本截断上限（默认 3000 token，最高 30000）。
- **搜索结果上限放开至 30**；默认 RAG 相似度阈值 0.4。
- **Provider 内置搜索模式**：部分供应商（如 Gemini）原生 grounding 搜索支持。

**Phase 6 — 工具管理与 MCP**

- **工具管理页**：内置 / 社区分类 + 开关控制 + 参数配置。工具名称持久化、displayName + configurableParams 完整 slang 多语言支持。
- **时间获取工具**：Agent 可主动获取当前时间。
- **快捷指令**：支持翻译、总结等预设快捷指令，多语言本地化。
- **MCP Server**：实现标准 SSE 握手传输协议，`diary_edit`、删除类工具 MCP 接入，SSE Endpoint URL 的绝对路径与 Query 路由解析兼容性增强。
- **工具开发规范**：`TOOL_DEV_GUIDE.md` 文档化。

**Phase 7 — 多工作区隔离架构**

- **Vault 完整架构**：多空间隔离存储、创建/删除（带手写安全核验）、切换与路径校验。
- **数据库隔离**：切换工作区时 drift AgentDatabase/AppDatabase 多实例安全管理。
- **跨端路径自愈**：VaultService 初始化时自动校验/修正跨端路径（registry 残留源设备绝对路径）。

**Phase 8 — 数据安全与备份同步**

- **DataArchiveManager 统一中枢**：全量快照 ZIP（含 Agent 多表 + RAG 向量 + 伙伴头像 + 全局配置 + 身份字典）的导出导入。
- **局域网快传增强**：mDNS 广播内置 `/avatar` 头像串流；雷达面板极简化去除 IP 显示。
- **S3 / WebDAV 云同步**：同步记录新增本地下载功能。
- **备份上限管理**：用户自定义最大备份数量（1~100，默认 20），超限自动清理旧备份。
- **Legacy 兼容导入**：旧版 v2.3.4 ZIP 格式自动识别并转换导入，导入前先恢复 RAG 数据再恢复日记，避免重复嵌入风暴。
- **导出优化**：跳过 SQLite WAL/SHM 日志文件（可达数百 MB），只打包实际引用的伙伴头像。

**Phase 9 — Agent UI 全线适配**

- **消息气泡头像**：支持 AI 伙伴头像（图片/emoji 降级渲染）与用户头像/昵称显示。
- **消息操作**：编辑 / 重新生成 / 复制按钮。
- **工具调用可视化**：执行耗时毫秒级显示、工具调用结果分组折叠卡片。
- **AI 建议补全**：Wrap 自适应布局气泡。流式回复时仅在用户处于底部才跟随滚动，上翻查看历史不打断。
- **费用弹窗**：双栏展示（📊 累计 API 消耗 + 📐 当前上下文大小）+ 上下文消息弹窗。
- **侧边栏重构**：两区布局（功能选项区 + 对话历史区）、流式监听会话列表、收缩模式、移动端右滑 Drawer。
- **点击标题快速切换模型**。
- **附件管理**：附件私有化持久存储（会话级联清理）+ 图片上传与多模态消息发送 + 设置入口。

**Phase 10 — 跨端 UI 与体验重构**

- **顶部标签栏**：记忆 / Agent 平级切换 + 滑动动画。日记↔Agent 页面全局渐变过渡动画。
- **自定义标题栏**：替代 Windows 原生标题栏。
- **Material Design 3 主题**：自定义色盘选择（HSL 滑块 + 动态 seedColor），默认白守专属浅蓝 `0xFF9AD4EA`。
- **移动端深度优化**：底栏常量映射导航、底边栏点击防误触（MIUI 虚假路由重置拦截）、Android 预测性返回栈保护、PopScope 精确退出控制。
- **四语国际化**：Agent 全模块 slang 覆盖，超 42 处硬编码中文国际化补全。
- **Windows 安装包**：Inno Setup 多语言安装包，支持自定义安装路径 + 升级时强制关闭旧进程。

**Phase 11 — 工程质量与代码治理**

- **SOLID 原则重构**：Agent UI 四轮大文件拆分（chat_message_bubble 等）；AgentChatNotifier 拆分（state / model_resolver / side_effects）；设置页与 Agent 主页面重构。
- **Agent 目录迁移**：`features/agent` → `agent` 统一归口。
- **路由治理**：首页路由 `/` → `/diary` 根除系统级拦截回弹；GoRouter 弹栈异常系列修复。
- **测试体系**：基础测试框架 + Agent 数据导入导出全链路自动化单元测试 + 搜索引擎 37 个单元测试 + 软件测试开发指南。

**Phase 12 — v3.0.0 稳定性收尾**

- **Windows 局域网崩溃根治**：将 mDNS 广播与发现从并发启动改为串行执行 + `Completer` 异步队列锁，彻底消除 UDP 5353 端口竞争。
- **AI 供应商默认关闭**：所有系统内置供应商 `isEnabled: false`，老用户一次性静默迁移（`migrated_disable_empty_providers_v3`）。
- **Windows 安装器加固**：`DisableDirPage=no` 开放安装路径选择；`CloseApplications=force` 强制终止旧进程避免 DLL 锁死。
- **总结生成提升至单例**：切换页面不再中断批量生成任务。
- **多项内存泄漏与并发崩溃修复**：embedding disposed 根治、RAG 高频迁移 SQLite 并发异常、工作区切换数据库多实例化、Gemini Tool 对齐断裂、FTS5 语法注入防御等。
