##### v1.x 版本记录

###### v1.1.0 「全平台响应式与云同步」 _(2026-02-22)_

> **里程碑**：白守完成了全平台的 UI 与体验重构，重写了 AI 底层并上线了多协议数据同步。

- **云同步上线**：新增支持 S3 (MinIO/COS) 与 WebDAV 多协议同步，打破设备孤岛。
- **响应式重构**：重写主架构布局，根据屏幕宽度自动切换侧边栏/底栏模式，完美适配平板与折叠屏。
- **AI 客户端与配置重构**：采用工厂模式重构底层 API 请求架构支持流式输出；重置全部 AI 设置面板，支持独立控制模型开关。
- **品牌重塑**：完成 `BaiShou` 全局品牌名替换与图标换新，解耦数据同步与局域网传输设置页。

###### v1.0.0 「全平台工程化与 AI 赋能」 _(2026-02-18 ~ 2026-02-19)_

> **里程碑**：白守实现了从"个人玩具"到"全平台可用工具"的跨越，并正式奠定 AI 核心体验。

- **品牌确立**：完成全线资源文件的更名，确立「纯白誓约，守护一生」的核心愿景。
- **AI 总结逻辑与配置**：实现「周/月/季/年」四维进阶压缩算法；实现自定义 AI 配置，原生支持 Gemini、Claude 及 OpenAI 兼容协议。
- **CI/CD 自动化**：建立 GitHub Actions 构建管线，支持多平台（Windows, Linux, Android）Release 自动打包；引入 Inno Setup 安装包与 ProGuard 混淆保护。
- **功能侧翼展开**：局域网秒传、多主题切换、数据打包导出与快照还原保护逻辑上线。
