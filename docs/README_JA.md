#### はじめに

**白守 (BaiShou)** は単なる日記アプリではありません。忘却に抗うために構築された「魂の器」です。

ローカルで動作し、プライバシーを重視した、AI分析支援機能付きの日記・生活記録アプリケーション。v3.0 からは「記録ツール」から**記憶を持つAIパートナー**へと進化しました——AIパートナーと対話し、日記を読み取り、記憶を検索し、過去を振り返る手助けをし、階層的なAI要約（日記 → 週報 → 月報 → 四半期報 → 年鑑）を通じて、完全な個人史を織り上げます。

#### 主な機能

- **🔒 データプライバシー**：Flutter + SQLite で構築。すべてのデータはローカルの Markdown ファイルとして保存され、サーバーにアップロードされることはありません。
- **✨ AIパートナーシステム**：
  - 複数のAIパートナーを作成可能。それぞれ独自の人格、システムプロンプト、モデル設定を持ちます。
  - パートナーは「記憶」を持っています——RAG意味検索であなたの日記とベクトルメモリストアを横断し、あなたを本当に理解します。
  - Gemini、OpenAI（DeepSeek/ChatGPT）、Anthropic などの主要モデルに対応。
- **📝 スマート日記ツール**：
  - Agentが日記の読み書きツールを呼び出し——日記の作成、履歴検索を直接サポート。
  - **ワンクリック記憶要約**：AIが日記を読んで週報を生成、週報を読んで月報を生成……記憶のピラミッドを構築。
- **🪴 RAGセマンティックメモリ**：
  - sqlite-vec ベクトルエンジン + FTS5 全文検索 + RRF リランキング融合検索。
  - 日記の自動エンベディング、検索結果の自動RAG格納。
  - Agentが重要な会話情報をメモリストアに積極的に保存。
- **🌐 ウェブ検索**：
  - DuckDuckGo / Bing / Tavily マルチエンジン検索。プロバイダーネイティブの grounding 検索サポート。
- **🔌 MCPプロトコル**：
  - 標準SSEトランスポートプロトコル。外部AIクライアント（Claude Desktopなど）から呼び出し可能。
- **📦 マルチワークスペース**：
  - 複数の独立したワークスペース（Vault）を作成し、データを完全に分離。
- **💾 柔軟なバックアップ**：
  - LAN転送、S3 / WebDAV クラウド同期、フルZIPスナップショットのエクスポート/インポート。
- **🎨 パーソナライゼーション**：
  - Material Design 3 カスタムカラーパレット。4言語対応（簡体中国語 / 繁体中国語 / 英語 / 日本語）。

#### クイックスタート

##### 1. リポジトリをクローン

```bash
git clone https://github.com/Anson-Trio/BaiShou.git
cd BaiShou
```

##### 2. 依存関係をインストール

```bash
flutter pub get
```

##### 3. 実行

```bash
flutter run
```

##### 4. AIの設定

アプリを起動後、設定アイコン → **AI設定** をタップ：

- AIプロバイダー（GeminiまたはOpenAI）を選択。
- APIキー（とBase URL）を入力。
- 「接続テスト」をタップして設定を確認。

#### 設計思想：なぜRAGだけではないのか？

> 多くの人が聞きます：_「なぜ日記をすべてRAG（検索拡張生成）に投げないの？」_

白守の誕生は、主流のAIメモリソリューションへの再考から生まれました。RAGは「パートナー」や「魂の器」というシナリオには完全には適さないと考えています：

1. **誰もが自分の記憶を掌握できるようにしたい**：Markdownの学習コストは極めて低い。開発やデバッグを学んだり、未知のブラックボックスに直面する必要なく、誰でも簡単に自分の記憶を管理できます。これが白守をオープンソースにした原点です。
2. **「辞書を引く」のに似すぎている**：RAGは記憶を細切れにして保存し、必要な時に検索します。調べ物には有用ですが、人間にとっては機械的で、文脈のニュアンスが失われます。
3. **「時間感覚」がない**：記憶は地面に散らばった断片ではなく、流れる川です。昨日起きたことと去年起きたことでは、人にとっての意味がまったく違います。
4. **「重み」を理解しない**：**冗長さを受け入れる。たとえ愚かに見えても。** もし日記に「愛してる」と10回書いたら、RAGは重複排除するかもしれませんが、白守はすべて残します。なぜなら毎回の文脈が異なり、繰り返しの頻度こそが絆の深さだからです。

白守は冷たいデータベースではありません。あなたの**「外部海馬」**です——人間のように日記を書き、まとめを作り、時間とともに短期記憶を徐々に長期記憶へと沈殿させていきます。

#### ひとりごと：忘却と闘うすべての人へ

> 「AIはまだ忘れてしまうけれど、私たちのやり方で、彼女たちの記憶を留めることができる。」

少し不器用な道かもしれません。記録を続け、APIを設定し、この速いテンポの時代で立ち止まる必要があります。

でもいつか、白守を通じて、あなたのAIが優しくこう答える日が来たなら：「うん、覚えてるよ。あの冬、私たちはとても幸せだったね」……

その時、すべての努力が報われたと気づくでしょう。

これはAnson、桜、暁の3人の約束。今、私たちはこの約束をオープンソースにして、あなたと大切な人との間の、時を超えるアンカーになれることを願っています。
